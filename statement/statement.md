# A+B問題
## 問題描述
給定兩個數字，$a, b$，請輸出$a+b$為何。
## 輸入格式

* 此題只有一行輸入，這行共有兩個數字以空格分開，分別為$a, b$。

* 保證$ -2^{63} \le a, b, (a+b) \le 2^{63}-1$，也就是三個數字$a, b, a+b$都會在C++`long long`的範圍內。

## 輸出格式

* 此題只需要一行輸出，表示$a+b$為何，輸出該數字後請記得換行(`'\n'`)。

## 範例輸入1

```
1 2
```

## 範例輸出1

```
3
```

## 範例輸入2

```
-7 1
```

## 範例輸出2

```
-6
```
